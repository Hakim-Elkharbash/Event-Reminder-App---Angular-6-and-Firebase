Event Reminder..

This app was developed to add, manage and send reminders of events, the app has several features and functions, and it was developed using Angular 6 and Firebase.

Avaliabe on Github.

Developed by: Hakim Elkharbash @ 2018.
   


- Functions & Features..

  1. Adding new events with GPS location and start/end date and time.
  2. Handeling all kind of input errors.
  3. Show notifications for app operations and actions.
  4. Show event details such as: duration and GPS location on the map.
  5. Update (Going/Not Going) status for all events.
  6. Managing three differentstauts (Happing Now - Comming - Completed) accourding to evant date/time.
  7. Delete specific evants or all events together after showing confirmation message.
  8. Apply three diffrent Angular 6 animations.
  9. Show loading spinner while pulling the events informations.
  10. Show/Hide events.
  11. Send email reminders.
  12. Add event details to these calenders: iCalender, Google, Yahoo and Outlook.
  13. Apply searching and filtering to get certain events.
  14. Disable search, filter and delete, if there are no events.
  15. Show welcome message,if there are no events.
